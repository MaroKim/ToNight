|< GIT bash 명령어 >									   |
|											   |
| $git init										   |
| : 현재 위치한 폴더에 .git을 생성하도록 한다						   |
| : .git가 생성된 위치에 있는 파일이나 디렉토리가 git에 올라가게 된다.			   |
|											   |
+------------------------------------------------------------------------------------------+
| < Git 설치 후 초기 환경 설정 (~/.gitconfig 파일 설정) >				   |
|  1. 작업자 이름 설정									   |
|	$git config --global user.name [USER이름]					   |
|											   |
|  2. 작업자 email 설정									   |
|	$git config --global user.email [USER 이메일]					   |
|											   |
|  3. 설정값 확인									   |
|	$git config --global --list							   |
+------------------------------------------------------------------------------------------+
|  리눅스 쉘 상에서도 .git을 생성한 디렉토리를 기준으로 다음과 같은 명령어가 적용된다.     |
|											   |
+------------------------------------------------------------------------------------------|
|  $git status										   |
|  : 새로운 파일이나 변경된 파일이 있을 경우, 상태메시지를 출력				   |
|											   |
|											   |	|											   |
|  $git diff										   |
|  : 파일이 수정되었을 경우에 수정된 부분을 출력하여 보여준다.				   |
|											   |
|											   |
|  $git add [Directory | FileName]							   |
|  : 새롭게 생성한 디렉토리나 파일을 추가해준다(commit하기 전에 항상 add가 우선한다)	   |
|											   | 
|											   |
|  $git status										   |
|  : 다시 status명령을 통해 확인해보면, 변경사항을 잡아주고 있음을 볼 수 있다 		   |
|											   |
|											   |
|  $git commit										   |
|  : 변경사항을 commit하기위해 commit명령어를 수행하게 되면, 리눅스 vi창이 나오는데 i단축키| |  를 통해 commit message를 입력하고, :wq 통해 완료하면 commit 완료			   |
|											   |
|  $git remote -v									   |
|  : <remote명>이 설정되어 있는 정보를 출력해 준다					   |
|											   |
|											   |
|  $git remote add <remote 명> <repository URL>  					   |
|  : 해당디렉토리에 <remote명>으로 repository URL을 저장하고 추가한다			   |
|  : 앞으로 gitHub에 업로드 시, URL입력 없이 remote명으로 해결가능하다			   |
|											   |
|											   |
|  $git remote push <remote 명> <branch 명> 	(branch 명은 master|slave 둘 중 하나)	   |
|  : remote명에 저장된 URL주소로 해당 경로에 commit된 파일 또는 디렉토리를 업로드한다	   |
|											   |
|											   |	|   $git remote pull <remote명> <branch 명>						   | |  : remote명에 저장된 URL주소로부터 파일과 디렉토리를 다운로드 한다			   |
